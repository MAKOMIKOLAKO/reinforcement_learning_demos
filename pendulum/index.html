How this differs from CartPole

PPO learns two things at once: a policy (the actor, which picks the torque you see applied here) and a value function (the critic, which estimates how good a state is). Only the actor is exported to these checkpoints — the critic never runs at inference time, here or in the real world.

What the critic buys you is training stability: it gives PPO a low-variance baseline to judge actions against, whereas REINFORCE (the CartPole demo on this site) learns from raw episode returns alone. In practice that means this checkpoint slider should sweep from flailing to competent swing-up-and-balance more smoothly and consistently than CartPole's REINFORCE progression, which tends to be noisier checkpoint-to-checkpoint.

Physics

Reimplements Gym's Pendulum-v1 exactly: gravity 10.0, mass 1.0, length 1.0, 0.05s timestep, torque clipped to ±2.0 N·m, angular velocity clipped to ±8.0 rad/s. Episodes run 200 steps and then auto-restart from a random initial angle and angular velocity, matching Gym's reset distribution.

Training happens offline; this page only runs the actor network's forward pass on your device.
